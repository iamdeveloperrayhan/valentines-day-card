# ❤️ Valentine's Day Card

A romantic and interactive Valentine's Day greeting webpage built using HTML, CSS, and JavaScript. This project displays heartfelt messages inside a unique scrollable card interface, making it a fun frontend project and a great addition to any web development portfolio.

## ✨ Features

- Interactive Love Messages
- Scrollable Card Interface
- Responsive Design
- Clean & Romantic UI
- Smooth User Experience
- HTML5
- CSS3
- Vanilla JavaScript

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/iamdeveloperrayhan/valentines-day-card.git
```

Navigate to the project folder:

```bash
cd valentines-day-card
```

Open the `index.html` file in your browser.
